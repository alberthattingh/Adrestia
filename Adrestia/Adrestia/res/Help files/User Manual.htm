Getting started

 

Login

When you start the application, you will be greeted by a login screen. There is two empty fields, username and password.

Enter your username (usually a 4-digit code for example 1004), and then enter your assigned or chosen password and press the login button.

 

General

 

View Special events

On the main menu, select the Special Events tab.

Click the Show All Events button to view all current and previous events and their details

Click the Show Upcoming Events button to view all the events that are still to take place and their details

 

View messages

On the main menu, select the Messaging tab.

Select the type of user from which you want to view messages.

Select the person to/from which the messages are (in the combo box beneath ‘Recipient’).

Select whether you would like to see the messages you sent or the ones you received.

Finally, click the Load Messages button.

 

Send messages

On the main menu, select the Messaging tab.

Select the type of user from which you want to view messages.

Select the person to whom you want to send the message (at the bottom in the ‘New Message’ compartment)

Enter your message in the text box and click the Send button when you are finished.

 

Specifics Guide for Instructors

 

Edit an event

On the main menu, select an event in the grid and click the Edit Event button to edit the event. You will be taken to a new screen to change the details.

 

Delete an event

On the main menu, select an event in the grid and click the Delete Event button to delete it.

 

Create new event

On the main menu, click the New Event button. You will be taken to a new screen to enter the details.

First, select the date and time at which the event will take place.

Then specify the ticket costs for adults, children and pensioners.

Next you have the option to choose an existing venue for the event, or to create a new venue for the event.

After that you have the option to choose an existing event type, or to create a new event type.

Lastly, click the Add button to finish creating the event.

You can cancel the process at any time.

 

Create new event type

In the ‘Create a new event’ menu, there is an option to create a new event type. If you click that button you will be shown a screen where you can enter the description for your new event type.

Click the Add button to finalize your new event type.

You can cancel the process at any time.

 

Create a new venue

In the ‘Create a new event’ menu, there is an option to create a new venue. If you click that button you will be shown a screen where you can enter the description for your new event type.

After entering your description select the maximum number of seats the venue can accommodate.

Click the Add button to finalize your new venue.

You can cancel the process at any time.

 

Set time for lessons

In the main menu, select the Lessons tab.

You will be taken to a screen where, first you have to select the date for the lesson via the built-in calendar.

Next, choose a time for the lesson as well as the price that students have to pay to book the lesson.

Then enter a brief description for the lesson, and select the max number of students allowed to attend the lesson at a time.

You can see a preview of the lesson by clicking the Preview button.

Once you are happy you can press the Add Lesson button.

 

View lessons and booked students

In the main menu, select the Lessons tab.

You will be taken to a screen where all of your lessons is displayed in a grid.

From here you can edit or delete a lesson.

You can also see the students who have booked the selected lesson by clicking the Load Students button.

 

Edit a lesson

In the main menu, select the Lessons tab.

You will be taken to a screen where all of your lessons is displayed in a grid.

Select a lesson by clicking on it in the grid, and then press the Edit Lesson button.

 

Delete a lesson

In the main menu, select the Lessons tab.

You will be taken to a screen where all of your lessons is displayed in a grid.

Select a lesson by clicking on it in the grid, and then press the Delete Lesson button.

 

 

Specifics Guide for Students

 

Book a lesson

On the main menu, click the Book Lesson button to open the lesson booking menu.

You will be greeted by a screen that shows you a list of available lessons to book.

Select a lesson to see a short summary of the lesson.

If you want to book that lesson, simply click the Book button.

Your booked lesson will then be displayed in the My Booked Lesson section.

You can cancel the booking at any time.

 

View your booked lessons

On the main menu, click the Book Lesson button to open the lesson booking menu.

Click the Show Booked Lessons button.

 

Specifics Guide for Admins

 

Add credit to a student’s account

On the main menu, select the Students tab and press the Credit Student Account button.

You will see a screen where you can select the student ID with a combo box.

Next, enter his first name and last name in their respective text boxes.

Then edit the amount of credits to be added to his/her account and press Add Credit to finalize the addition of credits.

You can cancel at any time.

 

Add a new student

On the main menu, select the Students tab, then click on the New Student button.

Here you can enter the student’s ID, first name, last name, cell number, email address and set his/her password, each in their respective text box.

Once you entered all the student’s data, click the Add Student button.

You can cancel the process at any time.

 

Edit student details

On the main menu, select the Students tab.

You can search for the student using the search box, and then select him/her by clicking on them in the grid.

Next click on the Edit Student button.

Enter the Student ID in the first text box, and enter a new (or the same) password. Enter the password again to confirm spelling

Next, enter the first name, last name, cell number and email address of the student, each in their respective text boxes.

Finally click the Save button to save the changes made.

You can discard the changes at any time by pressing the Cancel button.

 

Add a new instructor

On the main menu, select the Instructors tab, then click on the New Instructor button.

Here you can enter the instructor’s ID, first name, last name, cell number, email address and set his/her password, each in their respective text box.

Once you entered all the instructor’s data, click the Add Instructor button.

You can cancel the process at any time.

 

Edit instructor details

On the main menu, select the Instructors tab.

You can search for the instructor using the search box, and then select him/her by clicking on them in the grid.

Next click on the Edit Instructor button.

Enter the Instructor ID in the first text box, and enter a new (or the same) password. Enter the password again to confirm spelling

Next, enter the first name, last name, cell number and email address of the instructor, each in their respective text boxes.

Finally click the Save button to save the changes made.

You can discard the changes at any time by pressing the Cancel button.

 

Generate Basic Report

On the main menu, select the Reporting tab.

Just choose if you want the weekly, monthly, yearly or all the reports by selecting one of the radio buttons on the right hand side, and the just click the Generate button.

 

Generate Custom Report

 On the main menu, select the Reporting tab.

 

Manage ticket sales

On the main menu, select the Ticket Sales tab.

First, select the event ID by using the combo box at the top.

Next, select the date and time of the ticket sale.

Then select the number of each ticket type sold.

The total cost will be displayed, and then you can click the Add Sale button to finalize the sale.

You can clear all the fields by clicking the Clear button.

 

Backup the database

On the main menu, select the Backup data tab.

That’s it!

For your convenience the database also automatically backs itself up once every month.
